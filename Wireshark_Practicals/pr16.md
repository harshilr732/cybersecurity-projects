# Practical 16: Wireshark Display Filters

## Aim
To use filters for efficient traffic analysis.

## Key Filters

### IP Filter

ip.addr == 192.168.1.10


### SYN Packets

tcp.flags.syn == 1 && tcp.flags.ack == 0


### DNS

dns


### HTTP

http


### ICMP

icmp


## Analysis
- Filters reduce noise  
- Enable focused investigation  

## Conclusion
Effective filtering is essential for packet analysis.
