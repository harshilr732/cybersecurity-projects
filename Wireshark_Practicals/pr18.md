# Practical 18: End-to-End Network Flow Analysis

## Aim
To trace a complete network interaction.

## Flow

DNS → TCP → TLS → Data → Termination

## Observations

### DNS
- Resolves domain  

### TCP
- Establishes connection  

### TLS
- Secures communication  

### Data
- Encrypted transfer  

### Termination
- FIN / RST  

## Analysis
- Multiple protocols work together  
- Visibility reduces after encryption  

## Security Perspective
Visible:
- DNS, TCP, TLS metadata  

Hidden:
- Payload data  

## Conclusion
Understanding full flow is critical for real-world network analysis.
