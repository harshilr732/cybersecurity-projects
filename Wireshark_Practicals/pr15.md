# Practical 15: Detection of C2 Beaconing Behavior

## Aim
To identify command-and-control communication patterns.

## Observations
- Repeated connections to same IP  
- Fixed intervals  
- Small packet sizes  

## Analysis
Normal:
- Irregular timing  

Beaconing:
- Regular intervals  
- Consistent size  

## Security Perspective
Indicators:
- Periodic traffic  
- Same destination  
- Low data volume  

## Conclusion
Behavioral patterns reveal C2 even with encryption.
