# Practical 17: TCP Stream Reconstruction

## Aim
To reconstruct full communication sessions.

## Procedure
Right-click packet → Follow → TCP Stream

## Observations
HTTP:
- Full request/response visible  
- Credentials visible  

HTTPS:
- Encrypted payload  

## Analysis
- Converts packets into readable session  
- Helps understand communication flow  

## Security Perspective
Use cases:
- Credential analysis  
- Session investigation  

## Conclusion
Stream reconstruction simplifies packet-level analysis.
